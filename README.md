# FlowMind 心流

一個幫助你記錄成長與管理情緒的個人日誌 PWA 應用。

## 功能特色

✨ **記錄完成事項**
- 記錄每日成就
- 標記展現的能力
- 上傳照片記錄

💭 **記錄情緒時刻**
- 4 種情緒分類:快樂感恩、焦慮失落、期待平靜、其他
- 記錄情緒強度
- 人地事天氣完整記錄
- 焦慮時提供行動建議

📖 **查看過往記錄**
- 時間軸模式
- 多種篩選方式
- 行動建議統計
- 編輯功能

📥 **匯入匯出**
- 支援 CSV 格式(Excel/Google Sheets)
- 支援 JSON 格式(完整備份)
- 智慧合併資料

💾 **資料保存**
- localStorage + IndexedDB 雙重備份
- 自動儲存機制
- 離線可用

## 使用方式

### 在電腦上:
1. 訪問 GitHub Pages 網址
2. 開始使用!

### 在手機上(PWA):
1. 用 Safari(iOS) 或 Chrome(Android) 開啟網址
2. 點擊「加入主畫面」
3. 就像原生 App 一樣使用

## 資料安全

- 所有資料都儲存在你的裝置本地
- 不會上傳到任何伺服器
- 完全離線可用
- 建議定期匯出備份

## 技術

- 純前端 PWA
- localStorage + IndexedDB
- 響應式設計
- 離線優先
